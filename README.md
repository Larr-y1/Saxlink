# SaxLink: The Elite Saxophone Marketplace

**SaxLink** is a world-class, luxury online marketplace that connects elite saxophonists with prestigious clients, including hotels, restaurants, event planners, and private individuals. The platform facilitates direct discovery, secure booking, and institutional-grade payment handling for high-stakes musical engagements.

## 💎 The Premium Amethyst Aesthetic

SaxLink features a unified, high-fidelity visual identity characterized by the **Obsidian and Radiant Amethyst** palette. The UI is designed to exude professional precision and high-end elegance, ensuring it attracts both world-class virtuosos and prestigious institutional clients.

- **Obsidian**: Deep, matte blacks for structural authority.
- **Radiant Amethyst**: Vibrant purple accents (Primary) for creative excellence.
- **Compact Architecture**: Optimized vertical rhythm and high-density layouts for a professional, balanced feel.
- **Institutional Polish**: Glassmorphism, 3D shadows, and micro-animations across all touchpoints.

## 🚀 Core Features

- **The Elite Registry (Browse)**: An immersive discovery experience featuring verified artist dossiers, high-fidelity media, and advanced technical filters.
- **Operational Manual (How It Works)**: A guided journey for both clients and musicians, utilizing a "Musical Staff" path concept and custom-animated SVG illustrations.
- **Premium Multi-Step Booking**: A sophisticated, multi-stage engagement protocol for secure performance requests.
- **The Artisan Console**: A private dashboard for musicians to manage active itineraries, financial ledgers, and portfolio resonance.
- **The Client Concierge**: A dedicated suite for institutional clients to manage escrowed payments and active engagements.
- **Secure Escrow Vault**: Integrated protection for both parties, ensuring honorariums are held safely until performance completion.
- **M-Pesa Integration**: Fast, reliable payouts directly to mobile money within 24 hours of completion.

## 🛠 Tech Stack

- **Framework**: [React 18](https://reactjs.org/) with [Vite](https://vitejs.dev/)
- **State Management**: [Zustand](https://github.com/pmndrs/zustand)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/) with [Shadcn UI](https://ui.shadcn.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Type Safety**: [Zod](https://zod.dev/) validation

## 📦 Getting Started

### Prerequisites

- Node.js (Latest LTS recommended)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/saxlink.git
   cd saxlink
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```
